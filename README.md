# template-starter

> my luminous SAO template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao starter my-project
```

### From git

```bash
sao p1ch-jp/template-starter my-project
```
